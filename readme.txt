Plugin to generate Security Headers and Caching on Class A++

== Description ==
Security Headers is a browser plugin that helps improve website security by checking the HTTP headers of the visited webpage. The plugin checks for common security headers, such as Content-Security-Policy, Strict-Transport-Security, and X-XSS-Protection, and provides suggestions for improving the security of the website. The plugin is easy to install and use, and it is recommended for anyone who wants to improve the security of their web browsing experience:
    - X-Powered-By
    - Content-Security-Policy
    - Strict-Transport-Security
    - X-Xss-Protection
    - X-Frame-Options
    - X-Content-Type-Options
    - Referrer-Policy
    - Permissions-Policy
    - Cache-Control

The "Security Headers & Caching" option is added under "Settings" menu.

== Installation ==
1. Create the directory security-headers in your '/wp-content/plugins/' directory
2. Upload all the plugin's file to the newly created directory
3. Activate the plugin through the 'Plugins' menu in WordPress
